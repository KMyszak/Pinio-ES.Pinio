# Instalacja dodatku

Aby korzystać z **Mosquitto MQTT**, należy mieć działającą instancję **Home Assistant**. Dodatek jest dostępny w *Sklepie dodatków*, a jego instalacja jest bardzo prosta.

---

1. Otwórz swój **Home Assistant** (np. *haos.app*)
2. Z paska bocznego wybierz:  
    `Ustawienia → Aplikacje → Zainstaluj aplikację`
3. Wyszukaj dodatek **Mosquitto broker**:  

    <img width="429" height="208" alt="obraz" src="https://github.com/user-attachments/assets/e93b0ce8-c671-4231-af46-b8c5376c95e0" />   

4. Wybierz dodatek i kliknij **Zainstaluj**   
5. Po instalacji zaznacz opcję **Obserwator** i kliknij **Uruchom**:    

    <img width="429" alt="obraz" src="https://github.com/user-attachments/assets/6830a320-d860-415a-8300-8c272d226c86" />      
    
6. Po uruchomieniu przejdź do sekcji [**Tworzenie użytkownika**](Tworzenie-uzytkownika.md)
