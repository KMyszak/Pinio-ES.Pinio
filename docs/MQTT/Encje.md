# Encje

Dla urządzeń **Pinio** (MicroPython + MQTT) dostępne jest 5 [encji](https://www.home-assistant.io/docs/configuration/entities_domains/) (*entities*):

!!! note "Sterowanie"
    **RELAY 1** - Wyjście 1 (GPIO20)  
    **RELAY 2** - Wyjście 2 (GPIO21)

!!! note "Sensory" 
    **IN 1** - Wejście cyfrowe 1 (GPIO18)   
    **IN 2** - Wejście cyfrowe 2 (GPIO19)  
    **1-Wire** - Wejście 1-Wire (bez podłączonego czujnika)
