# Port COM

**Port COM** to numer interfejsu komunikacyjnego, przez który komputer łączy się z płytką ESP za pomocą konwertera USB–UART. Warto zapamiętać jego numer, ponieważ podczas wgrywania firmware musisz wskazać dokładnie ten port, aby narzędzia takie jak `esptool` czy **Tasmotizer** mogły nawiązać połączenie z urządzeniem.

## Sprawdzaniu numeru Portu COM
