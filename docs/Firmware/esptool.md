# Narzędzie esptool

`esptool` to oficjalne narzędzie wiersza poleceń służące do wgrywania **firmware** na układy **ESP8266**. Działa bezpośrednio przez port UART i zapewnia pełną kontrolę nad procesem flashowania - umożliwia m.in. kasowanie pamięci, odczyt parametrów układu oraz przesyłanie plików `.bin`.

---

1. Przejdź na stronę:
    
    [https://github.com/espressif/esptool/releases :material-open-in-new:](https://github.com/espressif/esptool/releases)

2. Pobierz odpowiednią paczkę dla swojego systemu operacyjnego i wypakuj ją:

    <img width="715" alt="obraz" src="https://github.com/user-attachments/assets/a24460e9-2ab4-48d6-9f57-e26cf99e9310" />

!!! info ""

    Narzędzie należy uruchamiać z wypakowanego folderu, korzystając z okna *Wiersza poleceń* lub *PowerShell* otwartego w tym katalogu.
