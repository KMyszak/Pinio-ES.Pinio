# Panel diagnostyczny

Oprogramowanie **Tasmota** posiada dodatkowe encje, które są dostępne w oknie `Diagnostyka` panelu głównego urządzenia - domyślnie większość z nich jest wyłączona.

Panel pozwala na szybkie sprawdzenie bieżącego stanu urządzenia - dzięki niemu można łatwo uzyskać najważniejsze informacje o działaniu sprzętu.

---

| Parametr               | Opis                                                 |
|------------------------|------------------------------------------------------|
| **Firmware Version**   | Zainstalowana wersja Tasmota                         |
| **IP**                 | Adres IP urządzenia w sieci lokalnej                 |
| **Last Restart Time**  | Czas, jaki upłynął od ostatniego restartu urządzenia |
| **MQTT Connect Count** | Liczba połączeń MQTT od uruchomienia urządzenia      |
| **Restart Reason**     | Powód ostatniego restartu urządzenia                 |
| **RSSI**               | Siła sygnału Wi-Fi w procentach                      |
| **Signal**             | Siła sygnału Wi-Fi w dBm                             |
| **SSID**               | Nazwa sieci Wi-Fi                                    |
| **WiFi Connect Count** | Liczba połączeń Wi-Fi od uruchomienia urządzenia     |
