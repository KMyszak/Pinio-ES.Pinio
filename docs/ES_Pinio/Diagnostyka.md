# Panel diagnostyczny

Oprogramowanie **Tasmota** posiada dodatkowe encje, które znajdują się w panelu głównym urządzenia w&nbsp;oknie `Diagnostyka` - większość z nich jest wyłączonych. 

Panel pozwala na szybkie sprawdzenie bieżącego stanu urządzenia - dzięki niemu można w łatwy sposób uzyskać wiele kluczowych informacji.

| Parametr               | Opis                                             |
|------------------------|--------------------------------------------------|
| **Firmware Version**   | Zainstalowana wersja Tasmota                     |
| **IP**                 | Adres IP w sieci lokalnej                        |
| **Last Restart Time**  | Czas od ostatniego restartu urządzenia           |
| **MQTT Connect Count** | Liczba połączeń MQTT od uruchomienia urządzenia  |
| **Restart Reason**     | Powód restartu urządzenia                        |
| **RSSI**               | Siła sygnału Wi-Fi w procentach                  |
| **Signal**             | Siła sygnału Wi-Fi w dBm                         |
| **SSID**               | Nazwa sieci Wi-Fi                                |
| **WiFi Connect Count** | Liczba połączeń Wi-Fi od uruchomienia urządzenia |
